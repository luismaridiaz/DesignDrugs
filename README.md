# 🧬 Target Profiler – Frontend

Aplicación web para el descubrimiento racional de fármacos mediante la integración de datos genómicos, estructurales y farmacológicos.

## 🚀 Características

- Búsqueda de enfermedades y genes asociados mediante Open Targets Platform.
- Perfil completo de dianas: UniProt, propiedades fisicoquímicas, estructuras PDB (con visor 3D interactivo), actividades ChEMBL, fármacos conocidos y mecanismo molecular (IA).
- Diseño generativo de anticuerpos (stub con simulación local o integración con backend).
- Favoritos y exportación de perfiles en JSON.
- Interfaz responsive y moderna.
